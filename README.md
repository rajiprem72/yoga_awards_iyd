# Yoga Awards IYD 2026 Website

Official responsive website for **Yoga Awards IYD 2026**, celebrating the **12th International Yoga Day – 2026** and the **13th Year Award Ceremony**.

This website is designed to showcase the event, award categories, nomination details, eligibility rules, and online registration for participants.

---

## 🌟 Features

- Fully responsive design (Mobile / Tablet / Desktop)
- Beautiful hero banner for International Yoga Day
- Award categories section
- About the Awards section
- Terms, Conditions & Eligibility
- Google Form nomination integration
- Contact details section
- Professional gold-theme award layout
